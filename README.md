# Pcaps
Pcap Files parts used by other projects

To rebuild the original pcap files, run:

```bash
./git-setup.sh --merge 
```

To clean up the part files run:
```bash
./git-setup.sh --rm
```

To retrieve the part files:
```bash
./git-setup.sh --checkout
```
More details on how to use the `git-setup.sh` script, run:

```
./git-setup.sh --help
```
